Использовалась нейросеть deepseek, общее время работы около 30 часов

Игра запускается кнопкой "Начать охоту". Игра про альтернативную абстрактную сову. Она должна охотиться на мышей. Мыши могут прятаться, если видят сову. Есть кнопки, которыми можно подняться, есть кнопки которыми можно опуститься, без нажатия кнопки сова медленно снижается
